# 📄 Spreadsheet Form Builder & Pagination Automation

This project automates the process of transforming raw spreadsheet input into structured, paginated output forms.

It simulates a **form generation system** inside Google Sheets using Apps Script.

---

# 🎯 Objective

To automate:

* input data recording
* structured formatting
* grouped row merging
* pagination into printable form layouts

---

# 🔗 Workflow Overview

```text
User Input Sheet
        ↓
Data Recording
        ↓
Intermediate Formatting
        ↓
Pagination Engine
        ↓
Final Output Sheet (Form Layout)
```

---

# ⚙️ Key Features

* automatic form numbering
* input logging with timestamp
* dynamic row grouping & merging
* multi-page output generation
* header support for first and next pages
* layout formatting for printing

---

# 🧩 Use Cases

* generating printable forms
* structured reports from raw input
* invoice / checklist generation
* operational data formatting
* replacing manual formatting workflows
